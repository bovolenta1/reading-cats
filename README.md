# Reading Cats (Web)

Initial setup for the **Reading Cats** web app: a **Next.js (App Router)** project using **Tailwind CSS** with basic routing to kickstart navigation and UI layout.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Requirements
- Node.js **>= 20.9** (recommended: current LTS)
- npm (or pnpm/yarn if you prefer)

> If you're using Node 18, some packages (and newer Next versions) may warn about unsupported engines.

## Getting Started (Local)
1) Install dependencies:
```bash
npm install
npm run dev
```
